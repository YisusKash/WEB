# INCOLpartes - Repuestos y Autopartes 🚗💨

![Versión](https://img.shields.io/badge/version-1.0.0-blue)
![Licencia](https://img.shields.io/badge/license-MIT-green)

**INCOLpartes** es una plataforma web moderna y optimizada para la comercialización de repuestos automotrices en Colombia. Especializada en marcas como Hyundai, Kia, SsangYong y Chevrolet, ofrece una experiencia de usuario fluida con enfoque en la conversión directa vía WhatsApp.

---

## 🚀 Características Principales

* **Diseño Responsivo:** Interfaz adaptada para dispositivos móviles, tablets y escritorio.
* **Carrusel Infinito:** Exhibición dinámica de "Productos Nuevos" y "Top de la Semana" mediante JavaScript nativo.
* **Menú Hamburguesa Inteligente:** Navegación optimizada que se cierra automáticamente al hacer scroll o clic fuera.
* **Conversión Directa:** Botón flotante de WhatsApp y botones de cotización vinculados a mensajes personalizados.
* **Formulario de Contacto:** Integración con *Web3Forms* para recibir consultas directamente al correo electrónico.
* **SEO Optimizado:** Metadatos configurados para mejorar la visibilidad en buscadores.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica para mejor accesibilidad y SEO.
* **CSS3:** Diseño basado en variables personalizadas, Flexbox y CSS Grid.
* **JavaScript (Vanilla):** Lógica personalizada para el carrusel y el menú interactivo, sin dependencias pesadas.
* **FontAwesome:** Iconografía profesional.
* **Google Fonts:** Tipografía "Inter" para una lectura clara y moderna.
